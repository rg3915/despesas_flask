# Despesas_flask

Expense is a Flask App.

App feito com [Flask][0].

	$ git clone https://github.com/rg3915/despesas_flask.git
	$ virtualenv -p python3 despesas_flask
	$ cd despesas_flask
	$ source bin/activate
	$ pip install -r requirements.txt
	$ mkdir -p data/users
	$ python tests.py
	$ python despesas.py

http://127.0.0.1:5000/
